# User & Role CRUD


## Installation

Install all requred packages from npm
```bash
npm install
```

## Usage (on development)
```bash
npm start
```

## Production build
```bash
npm run build
```

## Lib used

- react
- react-router-dom
- redux-toolkit
- formik
- yup
- material-ui (mui)
